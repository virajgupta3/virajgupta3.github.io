One-time setup

This page publishes straight to your GitHub repo. It needs a key that proves you're you.

1. Open github.com → new fine-grained token
2. Token name: logbook · Expiration: 1 year
3. Repository access: Only select repositories → pick virajgupta3.github.io
4. Permissions → Repository permissions → Contents: Read and write (leave everything else at No access)
5. Generate, copy the token (starts with github_pat_), paste below.

It's saved only in this browser on this device. If you ever want to revoke it, delete the token on GitHub — takes effect instantly. You'll repeat this once on each device you publish from (e.g. your iPhone).
